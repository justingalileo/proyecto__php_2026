# Proyecto PHP

## Descripción
Este es un proyecto para dar los primeros pasos desarrollando en PHP. Mostrando un ¡Hola, Mundo!

## Prerequisitos
1. Debemos instalar apache2
sudo apt install apache 2

## Pasos para la ejecución
1. Crear una carpeta para el proyecto.
2. Crear una archivo index.php
3. Codificar el ¡Hola, Mundo!
4. Descargar y colocar en la carpte las imágenes a utilizar.
5. Subir el proyecto a GitHub.
